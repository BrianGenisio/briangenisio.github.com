In preparation for my [talk next Saturday](http://www.dayofdotnet.org/AnnArbor/Fall2008/Sessions.aspx#10), I am publishing the code that I use to help drive my WinForms application via NUnit tests. The idea is simple -- instantiate the form and use extension methods on the controls to find the controls and drive from there.   

This is a preliminary release. I will update this file as I find more functionality s needed. It turns out that you don't need much (besides some creativity) to automate WinForms UIs. This set of extension methods will help you get started.   

Download [WAX -- Winforms Automation Extensions](http://bilzmoude.googlepages.com/FormExtensions.cs)
